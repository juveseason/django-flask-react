# Microservices Web App with Django, Flask and React

follow this [course](https://www.youtube.com/watch?v=0iB5IPoTDts&t=4379s&ab_channel=freeCodeCamp.org)

## Start Django App

`cd admin`
`docker-compose up --build`

## Start Flask App

`cd main`
`docker-compose up --build`

## Start React App

`cd react-crud`
`npm start`
